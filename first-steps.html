Dashboard.
Dashboard.
The Projects tab will appear when you log in. It contains a list of your projects related to brand, product or keyword monitoring. In this tab you can also find:

  • The number of new mentions for each project.
  • The total number of mentions for each project.
  • The project creation date.
  • Buzz change - the percentage change in the number of mentions found in the last 7 days vs. previous 7 days.
  • Additional project settings.
Create a project.
Adding a project.
A project relates to your brand, product or subject. Different projects can be used to monitor different aspects of your business. Creating a new project is the most important part of web monitoring. Project configuration is vital to the quality of the results collected by Brand24 on the Internet. The options available during the project setup are divided into 3 tabs: Keywords, Sources and Notifications.

Keywords.
This is the most important section, which allows you to configure the keywords and phrases related to your brand or product or the subject you want to monitor. It contains the following boxes:

  • Project name - this is the name under which the project will appear in the Brand24 dashboard. Please note that the project name has no relation to the type and quality of data gathered by Brand24. You should make this name unique and easy to identify (for you) among different projects.
  • Keywords - A list of keywords which identify your brand, product or subject. Examples: Nike, Burger King, financing, diet.
  • Required keywords - A list of keywords that help to fine-tune the results by finding only those entries, thatt contain these keywords.
  • Excluded keywords - A list of keywords and phrases, that, like required keywords, helps narrow down the results or eliminate less valuable results. All entries that contain one of the excluded keywords will be eliminated from the search results in this Brand24 project.

Sources.
The next tab contains options that allow elimination of results from selected sources. You can provide a list of social media users whose mentions will not be gathered. This feature can be helpful if:

  • appear in the results
  • entries from your own fanpage on social media sites appear in the results
  • you need to eliminate entries from unwanted or less valuable sites

This way, you leave only valuable results that are useful for your business.

In case you delete a project accidentally, we are able to restore your project upon request.

Notifications and reports.
This tab allows you to configure notifications and reports, informing you about new mentions. You can change the notification frequency or add extra email addresses here.
Mentions.
Mentions.
The Mentions tab allows you to see and manage the results collected by Brand24. Results are automatically assigned to one of eight categories. The tab also lets you filter the results by date, sentiment (positive or negative) and additional keywords. You can also group the results here, so they are easier to analyze (especially when working in teams). Under the filter options, there are charts displaying the number of results and sentiment for each category.

Next to each result you can find the following options:

  • Mark - this option marks the result as read or dealt with - you can keep track of the results where you interacted with the author.
  • Source - displays information about the source of the selected entry. For results from social media sites such as Facebook or Youtube, you can see some info about the author of each entry: number of friends, location and more. For other entries this button will show the statistics of the site that they came from.
  • Group - allows you to assign an entry to one of the groups. Groups help gather similar entries and manage them.
  • Delete - deletes entries or all entries from certain domains or authors. The second option will block the domain or author - no more entries from them will appear in the search results.
  • Sentiment - set the sentiment manually for each result.
Analysis.
Analysis.
This tab contains a set of statistics, graphs and analyses for your project. Here you will find:

  • A list of the most influential social media users, who have discussed the monitored brand, product or subject
  • A list of the users who have discussed the monitored subject the most.
  • Category breakdown - the percentage share for each category in all of the results.
  • The most active sites - shows sites where a discussion of the monitored subject occurred most often.

The Analysis tab gives you access to the key domains and users, who create the web image of your brand or product. This knowledge can be used in your Internet marketing campaigns.
Sources
Sources.
This tab contains information about the sources of results collected by Brand24 for the selected project - a list of sites and social media users that mentioned the monitored subject, brand or product.

For the latter category (Authors) - you can see an overview of the information about social media users, such as the number of friends or followers and the number of results that come from the source.
Quotes
Quotes.
The Quotes tab presents a set of negative and positive mentions of your brand, product or subject. The presentation helps you see the real picture of the users' opinions. The results in Quotes tab can also be used in reports and presentations for your PR campaigns.
Groups
Groups.
The Groups feature helps to categorize and manage the results. You can create up to 20 groups within Brand24 account and move selected entries to a group.

Groups are especially important in cases of shared access to a Brand24 account. This way you can make specific people responsible for certain groups of results. For example, the IT related results are sent to the IT department group and customer service results to the customer service department group. All groups can be supervised by the PR department.
